# Udemy-clone-project

A static front-end clone of the Udemy homepage built using **HTML5 and CSS3**.
This project recreates the core layout and UI structure of the Udemy website to practice front-end development and responsive layout design.

## Project Overview

The website includes a navigation bar with a search feature, course categories, recommended courses, popular courses, topic suggestions, and a footer section. It mimics the structure of Udemy's homepage while remaining a fully static project.

This project helped improve my understanding of **HTML structure, CSS styling, layout building, and UI replication from real-world websites.**

## Features

* Navigation bar with search bar and icons
* Course categories menu
* Sale banner section
* Recommended courses section
* Topic recommendations section
* Most popular courses section
* Footer with useful links
* Course cards with ratings and pricing
* Clean UI inspired by Udemy

## Technologies Used

* HTML5
* CSS3
* Google Fonts
* Font Awesome Icons
